## 💎 Installation

1. Clone this repository :

```bash
git clone https://github.com/SupakunZ/Portfolio.git
```

2. Navigate to the project folder and install dependencies :

```
cd Portfolio
npm install
```

3. Launch the application in development mode :

```
npm run dev
```
